### Hi there, I'm Jim Mickelsson.

I am an IT Security Student at Yrkeshögskolan Enköping with a background in Technical Onboarding for B2B SaaS. 

My previous responsibilities included:
* **Account Provisioning:** Configured B2B SaaS accounts for organizations by selecting subscription tiers, assigning email servers, and setting up initial Role-Based Access Control (RBAC) via the administration interface.
* **User Onboarding:** Invited the primary administrative user for each organization, granting them the ability to independently invite and manage the rest of their users.
* **Template Engineering:** Designed custom email, survey, and event templates for the organizations using HTML and CSS.
* **Rendering Optimization:** Wrote email-client-specific code to resolve complex rendering inconsistencies across different platforms.
* **Dynamic Automation:** Built automated marketing templates using Mustache to ingest and format data from JSON feeds on behalf of the organizations.
* **Domain Authentication:** Instructed organizations on implementing SPF, DKIM, and DMARC protocols for secure email delivery.
* **Security Advocacy:** Promoted the adoption of Multi-Factor Authentication (MFA) to protect accounts handling users' Personally Identifiable Information (PII).
* **Privacy Compliance:** Advised organizations on GDPR data minimization best practices to ensure they only stored necessary contact details.
* **Platform Improvement:** Submitted detailed bug reports that led to implemented system enhancements.
